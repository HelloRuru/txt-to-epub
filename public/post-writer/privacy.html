隱私權政策

Post Writer - 社群貼文排版 Chrome 擴充功能

最後更新：2026 年 2 月 26 日

Post Writer 完全在你的瀏覽器本機運作，不會收集、傳輸或儲存任何個人資料。

資料收集

Post Writer 不會收集任何資料。你輸入的貼文內容只存在你的瀏覽器記憶體中，關閉側邊欄後即消失。

本機儲存

Post Writer 使用瀏覽器的 localStorage 儲存以下偏好設定：

  • 深色/淺色模式選擇
  • 平台顯示偏好
  • 區塊顯示偏好
  • 格式模式偏好

這些資料只存在你的電腦裡，不會傳到任何伺服器。你可以隨時在瀏覽器設定中清除。

權限用途

  • sidePanel：在瀏覽器側邊欄顯示排版工具介面
  • contextMenus：提供右鍵選單「用 Post Writer 排版」功能
  • activeTab：讀取目前分頁的網址，用於自動偵測社群平台（Facebook / Instagram / Threads）
  • tabs：監聽分頁切換事件，即時更新平台偵測結果

以上權限僅用於上述功能，不會讀取網頁內容或其他個人資訊。

第三方服務

Post Writer 不使用任何第三方分析工具、廣告追蹤或外部 API。所有功能都在本機完成。

資料傳輸

Post Writer 不會向任何外部伺服器傳送資料。沒有 API 呼叫、沒有遙測、沒有使用紀錄回報。

兒童隱私

Post Writer 不會收集任何人的資料，包含 13 歲以下兒童。

政策變更

如果隱私權政策有更新，會在此頁面公告。重大變更會在擴充功能更新說明中通知。
