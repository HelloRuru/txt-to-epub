# crengine.wasm

This file is required but too large (~3.68MB binary) to download via WebFetch.

Download manually from:
https://github.com/bigbag/epub-to-xtc-converter/raw/main/web/crengine.wasm

Or use curl:
```bash
curl -sL "https://raw.githubusercontent.com/bigbag/epub-to-xtc-converter/main/web/crengine.wasm" -o crengine.wasm
```

Place the file in this directory alongside crengine.js.
